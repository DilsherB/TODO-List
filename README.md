# 📖 TODO Items' List <a name="about-project"></a>

> this is a project to add, delete and update tasks to do.

## 🛠 Built With <a name="built-with"> </a>

> HTML 5 , css3, javascript ES6, NPM and webpack.

### Tech Stack <a name="tech-stack"></a>

> HTML 5 , css3, javascript ES6, NPM and webpack

  <ol><h2>You can:</h2>

 <li>Update task</li>
